# Portfólio (projeto)

Este repositório contém um pequeno site de portfólio estático.

## Estrutura do projeto

- index.html — página principal (HTML5)
- css/
  - style.css — estilos do projeto (CSS3)
- js/
  - script.js — comportamentos e interações (JavaScript)
- img/ — imagens e assets do site

## Tecnologias utilizadas

- HTML5
  - Estrutura semântica das páginas e conteúdo principal.
- CSS3
  - Estilos responsivos, layout e design visual. Arquivo principal: `css/style.css`.
- JavaScript (vanilla)
  - Interatividade, manipulação do DOM e comportamentos. Arquivo principal: `js/script.js`.
- Assets
  - Imagens e ícones colocados na pasta `img/`.

## Como executar

1. Abra `index.html` no seu navegador preferido (duplo clique ou via "Abrir arquivo").
2. Opcional: rodar um servidor local (recomendado para desenvolvimento):

```bash
# com Python 3.x
python3 -m http.server 8000
# então abra http://localhost:8000
```
